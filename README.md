# Jamba-Extras

Requires Jamba

 

Adds a broker button for Broker Bars.
Simple click to open the main Jamba addon.
 

Adds lowered warning noises. (-10, -20, -30db)
Can and will be adding more warning sounds for use.  

J-Warning
J-Sonar
J-Wilhelm
J-Fatality
